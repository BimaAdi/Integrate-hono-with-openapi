# Integrate Hono with Openapi/Swagger

## Requirements
- Node >= 20
- npm or yarn or pnpm

## Installation
1. Install depedencies
```bash
npm install
# or 
yarn install
# or
pnpm install
```

2. Run server
```bash
npm run dev
# or 
yarn dev
# or
pnpm run dev
```

3. swagger doc will be available at [http://localhost:3000/ui](http://localhost:3000/ui)
